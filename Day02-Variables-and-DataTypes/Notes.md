# Day 2 Notes

## What is a Variable?

A variable is a named memory location used to store data.

Example:

```java
int age = 21;
```

---

## Variable Declaration

Declaration means creating a variable.

```java
int age;
```

---

## Variable Initialization

Initialization means assigning a value.

```java
age = 21;
```

or

```java
int age = 21;
```

---

## Primitive Data Types

| Data Type | Size | Example |
|-----------|------|---------|
| byte | 1 byte | 100 |
| short | 2 bytes | 20000 |
| int | 4 bytes | 100000 |
| long | 8 bytes | 9000000000L |
| float | 4 bytes | 12.5f |
| double | 8 bytes | 99.99 |
| char | 2 bytes | 'A' |
| boolean | JVM dependent | true / false |

---

## Non-Primitive Data Types

- String
- Arrays
- Classes
- Interfaces
- Objects

Example:

```java
String name = "Navajitha";
```

---

## Rules for Naming Variables

- Use meaningful names.
- Cannot start with a number.
- Cannot contain spaces.
- Cannot use Java keywords.
- Use camelCase.

Examples:

✔ studentName

✔ totalMarks

✔ mobileNumber

---

## Difference Between Declaration and Initialization

Declaration:

```java
int age;
```

Initialization:

```java
age = 21;
```

Both:

```java
int age = 21;
```
