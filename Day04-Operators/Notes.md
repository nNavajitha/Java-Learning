# Day 4 Notes

## What is an Operator?

An operator is a symbol that performs an operation on one or more operands.

Example:

```java
10 + 20
```

`+` is the operator.

---

## Arithmetic Operators

| Operator | Meaning |
|----------|---------|
| + | Addition |
| - | Subtraction |
| * | Multiplication |
| / | Division |
| % | Modulus (Remainder) |

---

## Relational Operators

Used to compare values.

- >
- <
- >=
- <=
- ==
- !=

Return type: **boolean**

---

## Logical Operators

| Operator | Meaning |
|----------|---------|
| && | Logical AND |
| \|\| | Logical OR |
| ! | Logical NOT |

---

## Assignment Operators

| Operator | Example |
|----------|---------|
| = | a = 10 |
| += | a += 5 |
| -= | a -= 3 |
| *= | a *= 2 |
| /= | a /= 4 |
| %= | a %= 2 |

---

## Unary Operators

- ++ Increment
- -- Decrement

### Pre Increment

```java
++a
```

Increment first, then use the value.

### Post Increment

```java
a++
```

Use the value first, then increment.

---

## Operator Precedence

Java evaluates operators based on precedence.

Example:

```java
10 + 5 * 2
```

Output:

```
20
```

because multiplication has higher precedence than addition.
