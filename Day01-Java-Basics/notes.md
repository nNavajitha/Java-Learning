# Day 1 Notes

## What is Programming?

Programming is the process of writing instructions that tell a computer what to do.

---

## What is Java?

Java is a high-level, object-oriented programming language developed by James Gosling at Sun Microsystems in 1995.

It follows the principle:

Write Once, Run Anywhere (WORA)

---

## Why Java?

- Platform Independent
- Object-Oriented
- Secure
- Robust
- Portable
- Multithreaded
- High Performance

---

## Java Architecture

Java Source Code (.java)

↓

Compiler (javac)

↓

Bytecode (.class)

↓

JVM

↓

Machine Code

↓

Output

---

## JDK

Java Development Kit

Contains:
- Compiler
- JRE
- JVM
- Development Tools

Used for developing Java applications.

---

## JRE

Java Runtime Environment

Contains:
- JVM
- Libraries

Used for running Java applications.

---

## JVM

Java Virtual Machine

Responsibilities:
- Executes Bytecode
- Converts Bytecode into Machine Code
- Manages Memory
- Performs Garbage Collection

---

## Compilation Process

HelloWorld.java

↓

javac HelloWorld.java

↓

HelloWorld.class

↓

java HelloWorld

↓

Output

---

## Structure of a Java Program

- Class
- main() Method
- Statements
- Output

Example:

class HelloWorld {

    public static void main(String[] args){

        System.out.println("Hello");

    }

}
